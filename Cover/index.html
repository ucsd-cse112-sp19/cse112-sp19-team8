This component has FIVE attributes.

number [required] specifies how many revealers to display in total. (Range from 1 to 4)

angle [required] specifies the rotation angle of the revealers. (Range from 0 to 90)

title [optional] specifies the title of the cover.

img [optional] specifies the center image of the cover.

bgColor [optional] specifies the background color of the cover.

Required Usage: <cover-reveal> should be used to wrap the <element> needs to be covered.
