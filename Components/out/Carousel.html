Carousel

Carousel

Creates a Carousel custom Web Element, capable of displaying multiple images. One image is displayed in front at a time. Two images are displayed on each side.

Constructor

new Carousel()

Fires when an instance of the carousel is created. Obtains left, middle, and right images for display.
Properties:
Name Type Description
number number Number of images.
imagefolder string The directory to grab images from.
interval number How frequently the front image is changed.
display string Changes display order. Can be "normal", "reverse", or "random".
blur bool Determines if images on the sides are blurred or not.

Example
<image-carousel number=8 imageFolder='img' interval=1500 display blur='false'> </image-carousel>
