TimelineItem

TimelineItem

Timeline Class

Constructor

new TimelineItem()

Fires whenever a timeline item is instantiated. Sets up framework for item to have a timestamp and text content.
Properties:
Name Type Description
timestamp String time for item component
position String whether on "left" or "right"
content String content for the timeline item.
