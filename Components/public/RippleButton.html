RippleButton

Creates a RippleButton custom Web Element, capable of creating multiple buttons of varying sizes and texts.

Constructor

new RippleButton()

Fires when an instance of the button is created.
Properties:
Name Type Description
content string The text displayed by the button.
class string The class name of the button.

How to Include into Project
<script src="http://localhost:8080/RippleButton/ripple_button.js" type="text/javascript"></script>
Example
<ripple-button class='yesbutton' content="Yes"></ripple-button>
<ripple-button class='nobutton' content="No"></ripple-button>
