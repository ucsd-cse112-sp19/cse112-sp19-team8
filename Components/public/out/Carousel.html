Class: Carousel

Carousel()

Creates a Carousel custom Web Element, capable of dispalying multiple images. One image is displayed in front at a time. Two images are displayed on each side.

Constructor

new Carousel()

Fires when an instance of the carousel is created. Obtains left, middle, and right images for display.
Properties:
Type Description
number 'number' - Number of images.
string 'imagefolder' - The directory to grab images from.
number 'interval' - How frequently the front image is changed.
string 'display' - Changes display order. Can be "normal", "reverse", or "random".
bool 'blur' - Determines if images on the sides are blurred or not.
