Revealer

Revealer

Used for creating stickers under cover component. When user clicks on 'enter' button, The stickers will slide up alone with the cover component. The class has a rotate field in style to create an overlay effect.

Constructor

new Revealer(element, options)

Fires when an instance of revealer is created. The Revealer rotates in different angles to create an overlay effect.
Parameters:
Name Type Description
element object The element to be wrapped
options object Structure that wraps field of Revealer(ex.angle)
